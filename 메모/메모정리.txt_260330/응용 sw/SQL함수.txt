SQL함수
- length() -> 몇 바이트인지 반환

- char_length() -> 글자수를 반환

- select upper(), ucase() -> 문자열을 대문자로 변환

- select lower() -> 문자열을 소문자로 변환

- TRIM('     a     ');  -> 양쪽 공백제거
- LTRIM ('     a     ');  -> 왼쪽 공백제거
- RTRIM('     a     ');  -> 오른쪽 공백제거
특정 문자 앞뒤 모두 제거
SELECT TRIM(BOTH 'X' FROM 'XXXHELLOWXXX'); -> 양쪽 x를 모두제거

- LPAD(), RPAD() -> 왼쪽, 오른쪽에 문자열을 채워넣음
- LPAD(문자열, 총길이, 채울문자) 
- select(hello,10,#); -> #####hello

- SUBSTING( ), SUBSTR( ), MID( )  -> 원하는 위치의 문자를 반환
- SUBSTRING(문자열, 시작위치, 길이) -> 시작위치에서 길이만큼 반환
// 음수를 쓸 경우 뒤에서부터 시작위치를 정함

- LEFT( ), RIGHT( ) -> 문자열에서 왼쪽/오른쪽부터 원하는 만큼 잘라서 반환
- LEFT(문자열,길이) -> 문자열에서 왼쪽부터 원하는 길이만큼 잘라서 반환

- instr ( ) -> 찾을 문자열이 시작되는 위치번호를 반환
- instr(전체문자열, 찾을문자열) -> 찾는 문자열이 시작되는 위치번호를 반환하고 없으면 0을 반환

- concat( ) -> 괄호 안에 문자열을 이어붙이는 함수
- concat('문자열1','문자열2',... ) ->문자열1과 문자열2 ..등을 이어붙임 // 숫자도 문자열로 변환해서 이어붙임

-- REPLACE(문자열, OLD, NEW)
-- 문자열 안에 있는 OLD 문자열을 찾아서 NEW 문자열로 치환

-- REPEAT(문자열, 횟수)
-- 같은 문자열을 여러번 반복해서 이어 붙이고 싶을 때
