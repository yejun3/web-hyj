DCL (data control language ) : 데이터 제어어

-데이터배이스에서 사용자 권한을 관리하는 SQL이다.
GRANT : 권한을 부여
REVOKE : 권한을 회수

DCL이 필요한 이유
-모든 사용자에게 모든 권한을 주면 위험해진다.

MYSQL에서는 사용자를 다음과 같이 구분한다.
'사용자명'@'접속자위치'
'teacher'@'localhost'

사용자 생성하는 법
create user '사용자명'@'접속자위치'
identified by '비밀번호';

 //'사용자명'@'%' -> 모든 위치에서 접속이 가능

모든 위치에서 접속이 가능한 'test_user' 사용자를 생성하세요
비밀번호는 1234입니다.

test_user의 권한을 확인하세요
