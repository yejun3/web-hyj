매서드
정의
    - 객체가 가지고 있는 기능(동작)을 실행하는 코드다.
    - 자바에서 메서드는 클래스 안에 작성

메서드 선언

접근 제한자 반환타입 메서드명(){
	//기능을 수행하는 코드
}

반환값이 없는 경우 타입으로 'void'를 쓰면 된다.

2. 호출 방법
    - 객체를 만든 뒤 점(.)을 찍고 메서드 이름을 작성한다.
        
  

void powerOn() { ... } //리턴값이 없는 메소드 선언
double divide(int x, int y) { ... } //double 타입 값을 리턴하는 메소드 선언
    - 소괄호 (())안에 필요한 값이 있으면 넣고, 없으면 비워둔다.(매개값)
