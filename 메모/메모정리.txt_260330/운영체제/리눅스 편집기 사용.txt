리눅스 편집기 사용
**vi 편집기 구조**


**모드 전환 흐름**
[명령모드] → i → [입력모드]
[입력모드] → ESC → [명령모드]
[명령모드] → : → [마지막행모드]
```
**기본 사용법**

- 파일열기

vi test.txt

파일이 존재할 때 -> 기본 파일을 열어서 수정 모드로 진입한다.
파일이 없을 때 -> 화면에는 빈 파일이 열리고 저장하는 순간 실제 파일이 생성된다.

- 입력모드 진입 : i키

- 입력 종료 : ESC키

- 저장 & 종료

:w    → 저장
:q    → 종료
:wq   → 저장 후 종료
:q!   → 강제 종료
:set number -> 행의 번호가 나옴
:set nonumber -> 행 번호가 숨겨짐
:%s/old/new/g -> 전체 파일에서 old문자열을 new로 치환
