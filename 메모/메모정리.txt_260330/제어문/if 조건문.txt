If else 
if (조건){
명령문  }
else{
명령문  }
: 조건을 만족하면 명령문을 실행 조건을 만족하지 않으면 else 뒤에 있는 명령문을 실행
// 조건이 모두 false일 경우 else문 뒤에 { } 부분이 실행함

else if 
if (조건){
명령문  }
else if (조건2){
명령문  }
// 두 개 이상 조건식을 사용할 때 사용
: 조건을 만족하면 명령문을 실행 조건을 만족하지 않으면 else if의 조건을 확인해서 true면 결과 출력

// else if는 여러번 사용 가능 else는 마지맏 1번만 사용가능
