while문 (반복문)
조건식으로 구성  반복할 횟수를 알고 있지 않은 경우에 사용
while 조건식이 false가 될 때까지 반복(무한반복)
for문과의 차이점 : for문은 반복횟수가 정해짐 while문은 정해지지 않음
// 하지만 while문도 반복횟수를 지정할 수도 있음
// 끝나는 조건을 먼저 생각하지않으면 무한반복되고 프로그램이 고장 날 수 있음
ex)
num = sc.nextInt();
while(num!=0){ // num의 데이터가 0ㅇ이 아니다.
                     // 0이 아닌 모든 수는 true이다
 num = sc.nextInt(); // 다시 값을 받겠다.
}
