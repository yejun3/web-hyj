변수 
int a = 50; // 변수를 선언하고 값을 초기화
: 변수 a라는 공간을 만들고 50이라는 값을 지정

//변수 선언, 변수 초기화

선언 : 메모리의 값을 저장하는 "공간"을 확보하는 과정
int a; // 나는 컴퓨터에게 숫자를 저장할 수 있는 a라는 이름의 공간을 만들라고 지시

초기화 : 값을 넣어준다.
a= 100; // a라는 공간에 100이라는 값을 넣어줌

출력
System.out.println( "a: " + a);
//문자열 + 변수(숫자) : 출력문에서 +는 문자열과 변수사이를 이어줌

데이터타입 
String: 문자열 "abcd " //  char: 문자형 ' A '
int : 정수형 "1" //  double : 실수형  "0.0"
boolean : 참과 거짓 // true는 참 false가 거짓

변수이름 규칙
대소문자구분, 길이제한은 없음
숫자로 시작할 수 없음
특수문자는 $ , _ 만 허용
(변수는 같은 공간에서는 똑같은 이름의 변수는 1번만 사용할 수는 있다.)

변수 표기법
카멜표기법 : 소문자로시작, 단어들의 시작점을 대문자 작성
파스칼 표기법: 대문자로 시작 단어들 시작점도 대문자 작성 (주로 class 작성시 사용)
스네이크 표기법 : 단어 사이에 언더바 활용 (주로 상수, 변하지않는 값에 사용)

정리
변수 선언: 변수값을 저장할 공간 확보 // string sr;
변수 초기화 : 변수에 데이터 입력 과정 // sr = "abcd";
리터럴 : 변수에 넣을 값 // abcd

명시적 초기화 : 변수의 선언과 초기화를 동시진행 // string sr = "abcd";


자료형의 크기
비트(bit) : 컴퓨터에서 정보를 표현하기 위한 최소단위 (0과1 로 표현)
바이트(byte): 1개의 byte는 8개의 bit로 이루어짐( 이진법으로 표기)
  0  |  0  |  0  | 0  | 0 | 0 | 0 | 0 
128 | 64 | 32 | 16 | 8 | 4 | 2 | 1 ( 1이면 해당 칸의 값을 더함 ex 8 =00001100)


자료형 long은 접미사 L을 붙임 (long pp= 8_000_000_000L)
Float는 접미사f가 붙음

부동소수점 표기법 
diuble num1 = 3.14e2; // 3.14 * (10^2)
