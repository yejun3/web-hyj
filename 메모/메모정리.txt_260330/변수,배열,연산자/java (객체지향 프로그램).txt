어떤 실행문이든 세미콜론( ; )을 사용 -> 컴퓨터에게 해당 줄은 기능이 여기서 끝난다.

java 개요 (JDK > JRE > JVM)
JDK : java를 사용한 프로그램을 개발할 때 필요한 소프트웨어 도구 기능
JRE : java를 사용한 프로그램을 실행을 위한 환경을 만드는 패키지
JVM : java를 실행시키기 위한 가상 머신

JVM: 작성 코드를 해독하고 컴퓨터에 실행시킬 수 있게 도와주는 가상 컴퓨터
자바코드 - 소스코드(확장자 .Javafile)를 컴파일 할 시 바이트코드(class.)로 변환 
JVM이 바이트코드로 실행
컴파일 : 소스코드를 컴퓨터가 이해하도록 번역하는 과정

변수: 어떤 값(data)를 저장할 수 있는 저장 공간
