윈도우 명령어 
파일 / 폴더 관리 명령어
mkdir 폴더명 : 폴더의 생성
cd 경로: 폴더의 이동
cd.. : 그 전 단계 경로로 이동
파일 목록 확인 
dir : 현재 폴더의 파일 목록을 출력

파일 삭제 
del 파일명

폴더 삭제 
rmdir 폴더명 /s /q
/s : 내부 파일까지 삭제 
/q : 확인 없이 삭제

ip주소 확인 
ipconfig 
sr
네트워크 연결확인 
ping 주소 :  인터넷 연결 테스트

실행중인 프로세스 확인
tasklist


현재 사용자 확인
whoami

입력한 내용을 그대로 출력하는 명령어 
echo 내용

파일 만들 때 사용하는 echo
echo 내용 > 파일명 : 파일을 만들고 안에 내용을 작성해준다.


echo 내용 >   : 덮어쓰기
echo 내용 >>  : 추가쓰기

dir*확장자
: 특정 확장자 상태만 보여줌

type : 파일에 들어있는 내용을 터미널에 출력하는 기능
-로그 확인 
- 설정파일 확인 
-간단한 텍스트 내용 확인 

type 파일명1, 파일명2, : 어러개의 파일을 이어서 출력 가능

type 파일명 > 파일명 2 : 파일명 1의 내용을 파일명 2 로 저장할 수 있다.

type 파일명 1, 파일명2 > 파일명 3 : 두 파일을 하나로 합친다.

type con : 키보드에서 값을 입력받아 출력
ctrl +z 누르고 엔터치면 종료

type con > 파일명 : 키보드에서 값을 입력받아 파일에 저장할 수있다.

환경변수
-시스템이나 프로그램이 사용하는 "설정 값"

기본적인 사용법
%변수명%
%cd% : 현재 작업중인 폴더 값이 들어있는 변수

%username% : 현재 로그인 사용자

%windir% : 윈도우 설치 경로 

%random% :0~32767 숫자 출력

wsl(Windows SusSystem for Linux)
윈도우 안에서 리눅스 환경을 실행할 수 있게 해주는 기능
윈도우를 그대로 쓰면서도 bash, Is, pwd, apt, git 과 같은 리눅스 명령어와 도구를 사용할 수 있다.
전통적인 가상머신이나 듀얼 부팅 없이 Windows에서 GNU/LInux 환경을 
직접 실행할 수 있다.

가상머신보다 가볍고 편하기 때문에 사용을 한다.

wsl을 통해 리눅스를 설치하게 되면 Linux 커널을 기반으로 만들어진  
Ubuntu가 설치가 된다.
확인 wsl -l -v
안 될 때 Ubuntu 설치하는 키 : wsl --install -d Ubuntu : 우분투 설치

sudo(Superuser Do)
-리눅스에서는 관리자 권한으로 실행해라 라는 뜻이다.

우분투에서는 일반 사용자가 아무 프로그램이나 시스템이 설치하거나 
바꾸지 못하게 막아둔다.
-실수로 보안이 망가지는 것을 방지
-보안 강화
-중요한 파일 보호
